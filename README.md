# JavaKing

這個 repository 記錄我學習《Java 王者歸來》時所撰寫的範例、習題，以及 AI 輔助提供的練習程式題目。

目前這個 repo 的主要用途不是展示大型專案，而是保存我的 Java 基礎學習歷程。內容包含 Java 基礎語法、流程控制、陣列、類別、物件、建構方法、封裝與 static 等主題。

## 學習目標

這個 repository 的目標是：

1. 建立 Java 基礎語法能力
2. 熟悉物件導向程式設計的基本概念
3. 透過章節範例與習題累積實作經驗
4. 訓練自己能夠從零寫出程式，而不是只看懂範例
5. 建立可追蹤、可回顧的程式學習紀錄

## 學習紀錄原則

我在這個 repo 中練習的不只是「把程式寫出來」，也包括：

1. 能否說明每一行程式在做什麼
2. 能否判斷變數屬於物件還是類別
3. 能否分辨 instance variable 與 static variable
4. 能否正確使用 constructor 初始化物件
5. 能否用 `this` 解決參數與成員變數同名問題
6. 能否修改題目條件後重新寫出程式
